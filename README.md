# KERP Fitness Frontend

基于 `React + TypeScript + TailwindCSS + Vite` 实现 4 个模块页面：

- 首页 `Home`
- 视频课程库 `Courses`
- 训练统计与记录 `Stats`
- 个人主页 `Profile`

## 运行

1. 安装依赖

```bash
npm install
```

2. 启动开发环境

```bash
npm run dev
```

3. 构建

```bash
npm run build
```

## 环境要求

- Node.js `>= 18`
- npm `>= 9`
